Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando <b> (não semântica)

Nesta outra temos um termo em destaque usando

<strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando <i> (não semântico)

Nesta frase temos um termo com ênfase usando <em> (semântico)

Texto Marcado

Podemos também criar um texto marcado usando <mark>

Outra maneira é usar parâmetro pra cor do CSS3 dentro da tag

Ou então colocar o seletor <style> CSS no cabeçalho chamando o mark e especificando propriedade e valor

Texto Grande e texto pequeno

Estamos criando um texto grande e um texto pequeno com a tag <small> nesse parágrafo. Atençao!! A tag <big> está OBSOLETA!

Texto Deletado e Texto Inserido

Podemos marcar um texto como deletado com a tag <del> para indicar que ele deve ser lido, mas não considerado.

Marcamos um texto como inserido para sublinhá-lo com <ins> (semântico) e dizer que ele foi inserido depois.
Não usamos mais a tag <u> (não semântica) pois está obsoleta.

Usamos ainda a tag <address> para destacar o endereço do autor.
Exemplo:
Fulano de Tal

Rua das Abóboras, 22, Centro, Rio de Janeiro

Texto Sobrescrito / Texto Subescrito

Para inserir coisas do tipo x5 + 20 usamos a tag <sup>

Para inserir coisas do tipo H2O usamos a tag <sub>
